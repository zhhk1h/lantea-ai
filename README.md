## 🌐 关于 Lantea

如今的互联网，就像一片充斥着冗余信息与信息茧房的汪洋。人们每天被海量的平庸观点、情绪垃圾和同质化数据所淹没。这就是现代版的**“熵增”**。

[**Lantea.ai**](https://lantea.ai) 旨在借助深度图谱与原生大语言模型，剥离信息表象，提供**反平庸**和**反共识**的深度思考，找出大模型和互联网中被忽略的信息。

---

## 📦 关于本仓库

本仓库作为 Lantea 计划的开源数据节点，核心使命如下：

1. **📑 深度报告存档**：用于存储经过人工审核和评估的 Lantea 深度分析报告。
2. **⚖️ 模型对照实验**：用于对比分析多个大模型（LLMs）的输出结果，揭示认知维度上的差异。
3. **📐 核心指标测算**：探索并建立衡量**“反平庸（Anti-Mediocrity）”**的量化指标体系。
